# 🔮 Destiny Desk

เว็บแอปดูไพ่ทาโรต์ ธีม **Art Nouveau** (พาสเทล + ลายเส้นสีทอง) ทำงานแบบ Responsive
สร้างด้วย **Next.js** โฮสต์ฟรีบน **Vercel + GitHub**

แอปทำหน้าที่ **จัดชุดข้อมูลไพ่** แล้วให้คุณกดปุ่มคัดลอก (Copy Prompt) ไปวางใน AI ฟรี
(เช่น ChatGPT, Gemini, Claude) เพื่อให้ AI ทำนายแบบ Storytelling — จึงประหยัดค่า Token

---

## ✨ ความสามารถ

- ไพ่ครบ **80 ใบ** (เลข 0–79) มีชื่ออังกฤษ + ความหมายไทยครบทุกใบ
  - รวมไพ่พิเศษ **The Well (เลข 22)** และ **The Artist (เลข 23)**
- เลือก **วัตถุประสงค์**: Love & Relationship, Career & Success, Self-Discovery
- เลือก **รูปแบบการวางไพ่**:
  - 1 ใบ (Daily Draw)
  - 3 ใบ (อดีต–ปัจจุบัน–อนาคต)
  - 10 ใบ (Celtic Cross) — คลิกที่ไพ่เพื่อเปิด มี Pop-up ความหมาย แล้วไพ่เปิดค้างไว้
- ปุ่ม **Copy Prompt สำหรับ AI** รวมชื่อ/คำถาม/ไพ่/วัตถุประสงค์เป็นชุดคำสั่งให้อัตโนมัติ
- **Tarot Journal** บันทึกประวัติการดูดวงไว้ใน LocalStorage (เครื่องคุณเท่านั้น เป็นส่วนตัว)
- มีคำเตือน (Disclaimer) ว่าเป็นแนวทางเพื่อความบันเทิงและสำรวจตัวเอง

## 🖥️ วิธีรันในเครื่อง (สำหรับมือใหม่ ทีละขั้น)

### 1) ติดตั้งโปรแกรมที่ต้องมี
- **Node.js** เวอร์ชัน 18 ขึ้นไป → ดาวน์โหลดที่ https://nodejs.org (เลือก LTS)
- **VS Code** → https://code.visualstudio.com

### 2) เปิดโปรเจกต์ใน VS Code
เปิด VS Code → เมนู **File → Open Folder** → เลือกโฟลเดอร์นี้ (`Tarot today`)

### 3) เปิด Terminal ใน VS Code
เมนู **Terminal → New Terminal** แล้วพิมพ์คำสั่งทีละบรรทัด:

```bash
npm install            # ติดตั้งไลบรารี (ทำครั้งเดียว)
npm run organize-images # จัดรูปไพ่เข้าโฟลเดอร์ให้อัตโนมัติ (ทำครั้งเดียว)
npm run dev            # เปิดเซิร์ฟเวอร์
```

### 4) เปิดเว็บ
เปิดเบราว์เซอร์ไปที่ 👉 **http://localhost:3000**

> หยุดเซิร์ฟเวอร์ด้วยการกด `Ctrl + C` ใน Terminal

## 🖼️ เรื่องรูปไพ่

- รูปไพ่ทั้งหมดอยู่ที่โฟลเดอร์ **`public/tarotimages/`** ตั้งชื่อไฟล์ตามเลขไพ่ เช่น `0.jpg`, `24.png`
- คำสั่ง `npm run organize-images` จะคัดลอกรูปเดิมของคุณ (ชื่อยาว ๆ) มาเปลี่ยนเป็นเลขไพ่ให้อัตโนมัติ
- ไพ่ใบใดยังไม่มีรูป จะแสดง **หลังไพ่** (ลายดาวสีทอง) แทน — ไม่ทำให้แอปพัง
- อยากเพิ่มรูปเอง: ตั้งชื่อไฟล์เป็น `<เลขไพ่>.jpg` หรือ `<เลขไพ่>.png` แล้ววางใน `public/tarotimages/`

ที่เหลือ (The Well, The Artist, King/Queen/Knight of Swords, Pentacles ทั้งหมด) วางรูปเพิ่มได้ภายหลัง

## 🚀 วิธีเอาขึ้นเว็บฟรีด้วย GitHub + Vercel

1. สร้างบัญชี GitHub (https://github.com) และ Vercel (https://vercel.com — ล็อกอินด้วย GitHub)
2. อัปโหลดโฟลเดอร์นี้ขึ้น GitHub (สร้าง Repository ใหม่แล้ว push โค้ด)
3. เข้า Vercel → **Add New → Project** → เลือก Repository ของคุณ → กด **Deploy**
4. รอสักครู่ Vercel จะให้ลิงก์เว็บฟรี เช่น `https://destiny-desk.vercel.app`

> รูปในโฟลเดอร์ `public/tarotimages/` จะถูกอัปขึ้นไปด้วย ให้รัน `npm run organize-images`
> ก่อน push เพื่อให้รูปอยู่ในโฟลเดอร์นั้นครบก่อน

├─ scripts/
│  └─ organize-images.mjs  # สคริปต์จัดรูปไพ่เข้าโฟลเดอร์อัตโนมัติ
├─ public/tarotimages/     # รูปไพ่ (0.jpg, 24.png, ...)
