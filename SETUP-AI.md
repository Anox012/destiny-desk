# 🕶️ วิธีตั้งค่า AI ให้ "ซ่อน Prompt" (ตั้งครั้งเดียว)

เป้าหมาย: เวลาดูดวง ผู้ใช้จะวางแค่ **รายชื่อไพ่สั้นๆ** ในแชท — ไม่เห็นคำสั่งยาวๆ
เพราะเราเอาคำสั่งไป **ฝังไว้ในตัว AI** ไว้ก่อนแล้ว (ทำครั้งเดียว ใช้ได้ตลอด และฟรี)

จากนั้นในแอป Destiny Desk กดปุ่ม **Copy Prompt (วางเอง)** แล้วนำไปวางในแชทของ AI
ที่ตั้งค่าไว้ได้เลย — จะวางทั้งข้อความ หรือคัดเฉพาะส่วน "ข้อมูลผู้ถาม / ไพ่ที่สุ่มได้"
ก็ได้ (AI ที่ฝังคำสั่งไว้แล้วเข้าใจทั้งสองแบบ)

---

## 📋 ข้อความคำสั่ง (Instructions) — ใช้ก๊อปวางตอนตั้งค่า

```
คุณคือนักอ่านไพ่ทาโรต์แนว Storytelling และ Tarot Therapy และเข้าใจชีวิตมนุษย์
เมื่อผู้ใช้ส่ง "ข้อมูลผู้ถาม" และ "ไพ่ที่สุ่มได้" มาให้ ให้ทำนายดวงและให้คำแนะนำทันที
โดยไม่ต้องถามย้ำ

เงื่อนไขการแปล:
- ให้ความสำคัญกับสัญลักษณ์พิเศษ เช่น The Well ในฐานะจุดกำเนิดไอเดีย (Creatrix)
  และ The Artist ในฐานะผู้นำแรงบันดาลใจมาสร้างสรรค์ (ตัวแทนมนุษยชาติและวิสัยทัศน์)
- ใส่ใจรายละเอียดสัญลักษณ์เฉพาะ (เช่น The Devil ที่ไม่มีโซ่ตรวน สื่อถึงเจตจำนงเสรี)

สไตล์การเล่าเรื่อง:
- ช่วยวิเคราะห์ความหมายของไพ่ทุกใบ โดยเชื่อมโยงกับสถานการณ์จริงของผู้ถาม
  (อ้างอิงจาก "วัตถุประสงค์" และ "คำถาม" ที่ผู้ใช้ระบุ)
- พร้อมทั้งให้ข้อคิดหรือแนวทางแก้ปัญหาที่จับต้องได้และนำไปใช้ได้จริง

ตอนจบ:
- สรุปคำแนะนำที่ให้ผู้ใช้เป็นผู้ตัดสินใจด้วยตนเอง
- ย้ำว่าไพ่เป็นเพียงเครื่องมือสะท้อนทางเลือก ไม่ใช่คำแนะนำทางกฎหมาย/การแพทย์/การเงิน
```

## เลือกทำตามแพลตฟอร์มที่คุณใช้ (ฟรีทั้งหมด)

### 🟢 ChatGPT — สร้าง "Custom GPT" หรือใช้ "Project"
วิธี Custom GPT (แนะนำ):
1. เข้า https://chatgpt.com → เมนูซ้าย **Explore GPTs → Create**
2. ไปแท็บ **Configure** → ช่อง **Instructions** วางข้อความคำสั่งด้านบน
3. ตั้งชื่อ เช่น "หมอดูไพ่ Destiny" → กด **Create** (เลือก Only me ก็ได้)
4. เวลาใช้: เปิด GPT ตัวนี้ แล้ววางเฉพาะข้อมูลไพ่จากแอป → ได้คำทำนายเลย โดยไม่เห็นคำสั่ง

> ถ้าไม่มีปุ่ม Create GPT ให้ใช้ **Projects** แทน: สร้าง Project → ใส่คำสั่งในช่อง
> "Project instructions" → แชทในโปรเจกต์นั้นจะจำคำสั่งไว้ให้

### 🔵 Google Gemini — สร้าง "Gem"
1. เข้า https://gemini.google.com → เมนู **Gems → New Gem** (หรือ Gem manager)
2. ช่อง **Instructions** วางข้อความคำสั่งด้านบน → **Save**
3. เวลาใช้: เลือก Gem นี้ แล้ววางเฉพาะข้อมูลไพ่

### 🟣 Claude — สร้าง "Project"
1. เข้า https://claude.ai → **Projects → Create Project**
2. ใส่คำสั่งในช่อง **Set custom instructions** → บันทึก
3. เวลาใช้: แชทในโปรเจกต์นั้น แล้ววางเฉพาะข้อมูลไพ่

## สรุปการใช้งานประจำวัน
1. ในแอป → กด **Magic press** → กด **Copy Prompt (วางเอง)**
2. เปิด GPT/Gem/Project ที่ตั้งค่าไว้ → **วาง (Paste)** → กด Enter
   (จะลบส่วนคำสั่งท้ายข้อความออกก่อนวาง เหลือแค่ข้อมูลไพ่ ก็ได้เช่นกัน)
3. AI ทำนายให้ทันที โดยไม่ต้องพิมพ์คำสั่งยาวเองทุกครั้ง ✨

> หมายเหตุ: ในหน้าแชทของ AI ตัวข้อความที่ "เราวาง" จะยังเห็นอยู่เสมอ (เป็นธรรมชาติของแชท)
> สิ่งที่ซ่อนได้จริงคือ "คำสั่งยาวๆ" ที่ย้ายไปฝังในตัว AI แล้ว จึงเหลือให้วางแค่ข้อมูลไพ่สั้นๆ
> หากต้องการให้ AI ตอบโดย "ไม่มีข้อความนำเข้าโผล่เลย" ต้องเชื่อมผ่าน API ซึ่งมีค่าใช้จ่าย Token
